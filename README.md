# 📞 Simple Phonebook System in C++

A simple console-based phonebook project using file handling in C++.

---

## 🚀 Features

- Add contact (name + number)
- Search contacts
- Delete contact after confirmation
- File-based storage (number.txt)

---

## 🧠 How it works

- All data is stored in a text file (`number.txt`)
- Program reads file line by line
- If a match is found, user can delete it
- Deletion is done by rewriting the file

---

## 🛠 Technologies

- C++
- File Handling (fstream)
- Basic console I/O
